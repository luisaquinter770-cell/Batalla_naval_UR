## Descripción
Este proyecto consiste en el desarrollo del juego clásico Batalla Naval. El jugador deberá adivinar la ubicación de los barcos enemigos en un tablero, realizando disparos estratégicos hasta derribarlos.

## Objetivo
Implementar la lógica básica del juego mediante programación, permitiendo al usuario interactuar con el tablero, realizar disparos y visualizar el estado de la partida en tiempo real.

## Funcionalidades esperadas
- Mostrar el tablero de juego  
- Permitir seleccionar casillas para disparar  
- Contar el número de barcos restantes  
- Contar el número de barcos derribados  
- Controlar el número de intentos disponibles  
- Leer la configuración del tablero desde un archivo CSV

## Tecnologías
- Lenguaje de programación: (por definir, por ejemplo Python)  
- Manejo de archivos CSV  
- Uso de estructuras como listas o matrices  

## Metodología de trabajo
Se trabajará en equipo utilizando un repositorio en GitHub para organizar el código, llevar control de versiones y distribuir las tareas entre los integrantes.
